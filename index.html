Правила при верстке

Начало верстки

Главный css файл style.css должен генериться самим последним, после добавление всех остальных стилей

Модальные окна

Для всплывающих окон можно использовать Bootstrap или Magnific Popup. Но желательно использовать Magnific Popup, так как у него меньше размер чем у bootstrap-а. Это будет отображаться на скорости загрузки сайта.

Шрифты для сайта

Шрифты сначала нужно искать в Google Fonts. А если там нет, то https://www.nomail.com.ua/font. Если и там нет, то у Кирилла)

Заголовок на сайте

На каждой странице сайта должен быть только один(!) заголовок h1. Если один уже есть, нужно использовать h2, h3 или h4.

Общие элементы сайта

Для кнопок(Например: Онлайн заявка, Заказать, Прочитать, и т.д), для ссылок, для пагинации и других общих элементов сайта стили нужно стараться прописывать код только один раз. Нужно стараться писать меньше кода. Не должно быть так:

#about .apply-btn {
color: #5456456;
margin: 5px;
}
#contact .apply-btn {
color: #5456456;
margin: 5px;
}
#feedback .apply-btn {
color: #5456456;
margin: 5px;
}
#blog .apply-btn {
color: #5456456;
margin: 5px;
}

Комментирование

Нужно везде стараться коментить!В js, в стилях, в php. Потому что нам иногда приходится разбираться в проектах друг друга, и чтобы быстро понять код, который там, это очень необходимо. Также когда долго не заходишь на сайт, сам забываешь что писал. И с этими комментами быстро вспомнишь
